Hermitage

Hermitage is an ongoing project that I work on in my free time. Hermitage is an open-world simulation game set in the highlands of Scotland with planned features like procedurally generated castles and emergent AI based on my university dissertation. The game has a large, open and performant terrain implemented using adaptive quadtree meshing and loaded from satellite imaging data. Uses the Irrlicht rendering engine, entt and more. The procedurally generated castles use rooms designed in Minecraft and are loaded as voxel models for simple but effective content creation.

CrowdPathfinding

CrowdPathfinding is the engineering portion of my university dissertation. The project is in two parts: a crowd simulation library and a UI that runs simulations in 2D using a provided configuration. The goal of the project was to investigate whether it was possible to use emergent behaviour to reduce the computational load of agent pathfinding in human crowds. Written in C++ along with Javascript proof-of-concepts, this project was the result of months of work and provides a library for other developers to use in their simulations. The library uses OpenGL compute shaders to simulate individual agents and the A* algorithm to calculate paths. There is also a demo included which uses Cairo to render sample environments.
