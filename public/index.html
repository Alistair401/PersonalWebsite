My projects:

Hermitage (WIP): An open world game using the Irrlicht rendering engine, entt and more. This project aims to procedurally generate castles and use emergent AI to make a unique feeling RPG. Implements continuous LOD terrain meshing to render large heightmaps using data from satellite imaging (demo). The procedurally generated castles use rooms designed using Minecraft for simple but effective content creation.

AssimpViewer: A model and skeletal animation viewer using OpenGL and Assimp. Performant GPU skinning is used to animate models (demo). This project reached a natural end once animation had been implemented succesfully. The process was documented on the project's GitHub page.

ECS: A simple but extensible implementation of an Entity Component System library written in modern C++. Uses principals described by Vittorio Romeo.

CrowdPathfinding: An investigation into using emergent behaviour to optimise human crowd pathfinding. This was the engineering side to my dissertation project. Written in C++ along with Javascript proof-of-concepts, this project was the result of months of work and provides a library for other developers to use in their simulations. The project uses OpenGL compute shaders to calculate interactions between entities. There is also a demo included which uses Cairo to render sample environments. The full dissertation is available here.
